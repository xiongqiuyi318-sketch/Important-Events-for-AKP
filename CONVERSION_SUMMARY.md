## 快速开始

### 部署网页应用

```bash
# 1. 安装依赖
pnpm install

# 2. 配置环境变量（.env 文件）
TARO_APP_SUPABASE_URL=your_url
TARO_APP_SUPABASE_ANON_KEY=your_key

# 3. 构建
pnpm build:h5

# 4. 部署 dist 目录到任何静态托管服务
```

### 部署微信小程序

```bash
# 1. 构建小程序
pnpm build:weapp

# 2. 使用微信开发者工具打开 dist 目录
# 3. 配置 AppID 并上传
```
